# GitHub Activity Viewer

This is a Flask-based web app that allows you to pull your own GitHub activity.

## Getting Started

### Prerequisites

- Python 3.x
- pip (Python package installer)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/your-repository.git
   cd your-repository
   ```
2. **Create a virtual enviorment:**
    ```bash
    # On Windows
    python -m venv venv

    # On macOS/Linux
    python3 -m venv venv
    ```
3. **Activate the virtual enviorment**
    ```bash
    # On Windows
    .\venv\Scripts\activate

    # On macOS/Linux
    source venv/bin/activate
    ```
